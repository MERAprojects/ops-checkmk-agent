OPS-SYSD
=========

What is ops-checkmk-agent?
------------------
ops-checkmk-agent is the repo for the OpenSwitch Check_mk agent.

What is the structure of the repository?
----------------------------------------
* agents - contains all check_mk agent scripts
* tests - contains all automated tests for checkmk-agent.
* docs - contains the documents associated with this repo.
